## 📋 Quick Deploy Steps (5 Minutes)

### 1. Upload & Setup Laravel (2 min)
```bash
# Upload project
sudo mkdir -p /nginx/myproject
cd /nginx/myproject
sudo git clone https://github.com/username/project.git .

# Quick setup
sudo chown -R forge:forge /nginx/myproject
sudo chmod -R 755 /nginx/myproject
sudo chmod -R 775 /nginx/myproject/storage /nginx/myproject/bootstrap/cache
composer install --no-dev --optimize-autoloader
cp .env.example .env && php artisan key:generate
```

### 2. Create Single Nginx Config (1 min)
```bash
sudo nano /etc/nginx/sites-available/myproject.conf
```

**Copy-Paste This Complete Config:**
```nginx
# Replace: myproject.com, /nginx/myproject
server {
    listen 443 ssl;
    server_name myproject.com www.myproject.com;
    root /nginx/myproject/public;
    index index.php;

    # SSL (Auto-added by Certbot)
    # ssl_certificate /etc/letsencrypt/live/myproject.com/fullchain.pem;
    # ssl_certificate_key /etc/letsencrypt/live/myproject.com/privkey.pem;

    # Laravel routing
    location / { try_files $uri $uri/ /index.php?$query_string; }

    # PHP processing
    location ~ \.php$ {
        include snippets/fastcgi-php.conf;
        fastcgi_pass unix:/var/run/php/php7.4-fpm.sock;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        include fastcgi_params;
    }

    # Security
    location ~ /\.(ht|env) { deny all; }
    
    # File uploads
    client_max_body_size 100M;
}

# HTTP redirect
server {
    listen 80;
    server_name myproject.com www.myproject.com;
    return 301 https://$host$request_uri;
}
```

### 3. Enable & SSL (2 min)
```bash
# Enable site
sudo ln -s /etc/nginx/sites-available/myproject.conf /etc/nginx/sites-enabled/
sudo nginx -t && sudo systemctl reload nginx

# Install SSL
sudo certbot --nginx -d myproject.com -d www.myproject.com
```

## ⚡ Ultra-Fast Template

### Single Command Deploy
```bash
# Replace variables and run
PROJECT_NAME="myapp"
DOMAIN="myapp.com"
GIT_REPO="https://github.com/user/repo.git"

# Complete deployment
sudo mkdir -p /nginx/$PROJECT_NAME && cd /nginx/$PROJECT_NAME
sudo git clone $GIT_REPO . 
sudo chown -R forge:forge /nginx/$PROJECT_NAME
sudo chmod -R 755 /nginx/$PROJECT_NAME && sudo chmod -R 775 /nginx/$PROJECT_NAME/storage /nginx/$PROJECT_NAME/bootstrap/cache
composer install --no-dev --optimize-autoloader
cp .env.example .env && php artisan key:generate

# Create nginx config
sudo tee /etc/nginx/sites-available/$PROJECT_NAME.conf > /dev/null <<EOF
server {
    listen 443 ssl;
    server_name $DOMAIN www.$DOMAIN;
    root /nginx/$PROJECT_NAME/public;
    index index.php;
    
    location / { try_files \$uri \$uri/ /index.php?\$query_string; }
    location ~ \.php$ {
        include snippets/fastcgi-php.conf;
        fastcgi_pass unix:/var/run/php/php7.4-fpm.sock;
        fastcgi_param SCRIPT_FILENAME \$document_root\$fastcgi_script_name;
        include fastcgi_params;
    }
    location ~ /\.(ht|env) { deny all; }
    client_max_body_size 100M;
}
server {
    listen 80;
    server_name $DOMAIN www.$DOMAIN;
    return 301 https://\$host\$request_uri;
}
EOF

# Enable and SSL
sudo ln -s /etc/nginx/sites-available/$PROJECT_NAME.conf /etc/nginx/sites-enabled/
sudo nginx -t && sudo systemctl reload nginx
sudo certbot --nginx -d $DOMAIN -d www.$DOMAIN
```

## 🔧 Common Quick Fixes

### Laravel Setup
```bash
# Database & Cache
php artisan migrate --force
php artisan config:cache && php artisan route:cache && php artisan view:cache

# Fix permissions
sudo chown -R forge:forge /nginx/myproject
sudo chmod -R 775 /nginx/myproject/storage /nginx/myproject/bootstrap/cache
```

### Nginx Issues
```bash
# Test & reload
sudo nginx -t && sudo systemctl reload nginx

# Check status
sudo systemctl status nginx php7.4-fpm

# Restart services
sudo systemctl restart nginx php7.4-fpm
```

### SSL Problems
```bash
# Force SSL renewal
sudo certbot renew --force-renewal

# Check certificates
sudo certbot certificates
```

## 🎯 Production-Ready Single Config

```nginx
# Production Laravel + Nginx Config
server {
    listen 443 ssl http2;
    server_name myproject.com www.myproject.com;
    root /nginx/myproject/public;
    index index.php;

    # SSL
    ssl_certificate /etc/letsencrypt/live/myproject.com/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/myproject.com/privkey.pem;
    include /etc/letsencrypt/options-ssl-nginx.conf;
    ssl_dhparam /etc/letsencrypt/ssl-dhparams.pem;

    # Security Headers
    add_header X-Frame-Options "SAMEORIGIN" always;
    add_header X-XSS-Protection "1; mode=block" always;
    add_header X-Content-Type-Options "nosniff" always;

    # Laravel
    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }

    # PHP
    location ~ \.php$ {
        include snippets/fastcgi-php.conf;
        fastcgi_pass unix:/var/run/php/php7.4-fpm.sock;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        include fastcgi_params;
        fastcgi_hide_header X-Powered-By;
    }

    # Security
    location ~ /\.(ht|env|git) { deny all; }
    location ~ /storage/.*\.php$ { deny all; }

    # Static files caching
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }

    # File uploads
    client_max_body_size 100M;
    client_body_timeout 300s;
}

# HTTP redirect
server {
    listen 80;
    server_name myproject.com www.myproject.com;
    return 301 https://$host$request_uri;
}
```

## ⚠️ Emergency Troubleshooting

### 502 Bad Gateway
```bash
sudo systemctl restart php7.4-fpm nginx
sudo tail -f /var/log/nginx/error.log
```

### 403 Forbidden
```bash
sudo chown -R forge:forge /nginx/myproject
sudo chmod -R 755 /nginx/myproject
```

### SSL Issues
```bash
sudo certbot --nginx -d domain.com --force-renewal
```

### Laravel 500 Error
```bash
php artisan config:clear && php artisan cache:clear
sudo chmod -R 775 /nginx/myproject/storage
```
